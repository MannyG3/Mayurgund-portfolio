# Mayurgund-portfolio

Japanese-themed personal portfolio for Mayur Gund, highlighting projects and contact links with a light and dark mode aesthetic.

Live site: https://mannyg3.github.io/Mayurgund-portfolio/

## Features

- Light and dark theme toggle with katana slice transition
- Hero section with Tokyo night and foggy torii imagery
- Projects grid with glassmorphism cards and high-contrast typography
- Contact section with neon hover accents and readable overlays
- Responsive layout for mobile and desktop

## Local development

From the repository root:

```bash
python3 -m http.server 8000
```

Open http://localhost:8000 in your browser.
